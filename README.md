
# Dialexis


**Dialexis** — peer-to-peer чат на Java с графическим интерфейсом (JavaFX) и поддержкой консольного режима. Прямое соединение между пользователями реализовано через gRPC.

---

## Описание


Dialexis — это учебный проект, реализующий чат между двумя пользователями без центрального сервера. Основной способ использования — современный графический интерфейс (JavaFX), также поддерживается консольный режим. Передача сообщений осуществляется по gRPC, что обеспечивает строгий контракт и типизацию.

---


## Возможности

- Современный графический интерфейс (JavaFX):
	- Ввод параметров подключения и имени пользователя через форму
	- Просмотр истории сообщений в отдельной панели
	- Удобное поле для ввода и отправки сообщений
	- Кнопки управления соединением (старт, подключение, отключение, выход)
	- Отображение системных статусов
- Прямое peer-to-peer соединение между двумя пользователями (gRPC)
- Отображение имени отправителя, времени и текста сообщения
- Запуск в режиме сервера (ожидание подключения) или клиента (подключение к peer)
- Валидация параметров и обработка ошибок подключения
- Поддержка консольного режима (CLI) для работы из терминала
- Модульные и интеграционные тесты

---

## Архитектура

- **UI (ConsoleUi):** Ввод/вывод сообщений, отображение системных событий.
- **Application (ChatService):** Логика отправки/приема сообщений, взаимодействие с транспортом.
- **Transport (gRPC):** Сервер и клиент для передачи сообщений, преобразование моделей.
- **Domain:** Модели сообщений и адресов, интерфейсы.

Подробнее: [ARCHITECTURE.md](docs/ARCHITECTURE.md)

## Установка и запуск

1. **Требования:** Java 17+, Gradle, интернет для загрузки зависимостей.
2. **Сборка:**
   ```sh
   cd code
   ./gradlew build
   ```

---

## Запуск графического интерфейса (GUI)

Основной способ использования:

```sh
./gradlew runGui
```

В открывшемся окне:
- Введите имя пользователя и порт для локального сервера
- (Опционально) укажите адрес и порт peer для подключения
- Нажмите **Start** для запуска сервера, **Connect** для подключения к peer
- Вводите сообщения в нижнем поле, отправляйте кнопкой или Enter

---

## Запуск в консольном режиме (CLI)

Для работы из терминала доступны следующие команды:

- Сервер (ожидание подключения):
  ```sh
  ./gradlew run --args="--name Alice --port 50051"
  ```
- Клиент (подключение к peer):
  ```sh
  ./gradlew run --args="--name Bob --port 50052 --connect-host 127.0.0.1 --connect-port 50051"
  ```
